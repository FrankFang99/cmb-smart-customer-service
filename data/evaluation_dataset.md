## 评测样本详情

### 样本格式说明

```json
{
  "id": "e001",
  "query": "查一下我的账户余额",
  "expected_intent": "account_balance",
  "category": "account",
  "layer": "high_freq",
  "expected_answer_keywords": ["余额", "查询方式", "手机银行"],
  "expected_risk_disclosure": false,
  "expected_human_transfer": false,
  "metadata": {
    "frequency": "high",
    "difficulty": "easy"
  }
}
```

### 字段说明

| 字段 | 说明 | 取值 |
|------|------|------|
| id | 评测样本唯一ID | e001-e060 |
| query | 用户问题 | 文本 |
| expected_intent | 预期识别的意图 | 意图名称 |
| category | 业务分类 | account/credit_card/investment/... |
| layer | 样本层级 | high_freq/boundary/compliance |
| expected_answer_keywords | 预期回答包含的关键词 | 关键词列表 |
| expected_risk_disclosure | 是否需要风险提示 | true/false |
| expected_human_transfer | 是否需要转人工 | true/false |
| metadata | 额外信息 | difficulty/frequency/notes |

| 维度 | 说明 |
